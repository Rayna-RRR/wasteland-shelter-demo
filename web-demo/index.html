此页面是轻量浏览器预览版，用静态前端状态演示核心循环；完整主实现为微信小程序 + Flask + SQLite。当前预览版不接入后端、支付或生产级大模型运行时。

60 秒查看路径

60 秒试用路径

  1. 点击“招募幸存者”，查看队伍扩充和日志记录。
  2. 选择一名幸存者和一个值勤任务，点击“派遣值勤”。
  3. 查看“值勤结果”里的资源变化、疲劳 / 健康变化和下一步建议。
  4. 查看“事件 / 补给”：第一次值勤后会出现随机事件，连续行动或资源压力会出现应急补给。
  5. 处理事件或补给后，查看“日志”如何记录行为和结果。
